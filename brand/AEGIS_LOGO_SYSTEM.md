# AEGIS Logo System

Status: final V3.0 commercial package artifact.

## Concept

The symbol combines a shield, signal aperture, and governance axis to express protection, intelligence, and controlled enterprise infrastructure.

## Variants

- Primary horizontal logo
- Icon-only mark
- Vertical lockup
- Small favicon mark
- Dark mode version
- Light mode version

## Rules

- Keep safe spacing equal to the icon width
- Do not stretch or rotate
- Do not place on low-contrast backgrounds
- Do not add unsupported certification marks
